    Mobile Panini Website

    This project is of a model of a website for a mobile restaurant, called "Mobile Panini". It features multiple pages which can be visited from each other using a navigation bar featured near the top of the page. Featured are pages for the homepage, a menu, and events and catering information. This was a project that focused on basic HTML elements and attributes, such as lists and hypertext links, and using them to create a multi-page site.

    Tri and Succeed Sports Coaches

    This project is of a model of a page for coaches of an athletic service called "Tri and Succeed Sports".     This project is a model for a website of an aquatic-themed internet-providing service called "Marlin Internet".     Red Ball Pizza Customer & Build

    This project is a model for a couple of pages for a pizza restaurant called "Red Ball Pizza". The customer page features a form for customer information for the purpose of delivery service by the restaurant. Inputting actual information into the boxes will prompt symbols and window alerts for the user. The build page features another form for the purpose of allowing a customer to custom-build their own pizza, including sections for size, crust, and toppings. This project introduced web forms and using them to collect information, as well as styles suited for forms, like placeholders.
